== Description ==
This plugin adds a small widget within each blog post to share blog posts on  Facebook, Twitter, Google Buzz, Digg, Surfpeople, and well over 100 more social bookmarking and sharing sites .

<h4>Try the Demo</h4>

If you're interested in seeing what a default installation of Social Share  has to offer, try out the Social Share  Test Blog! 

<a href="http://www.websaledomain.com/index.php?page=tool">Social Share  Test Blog </a>

== Installation ==

Installation is fairly simple

1. Upload `social-share-surfpeople.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
